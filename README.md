Старые версии  доступны в разделе [Releases](https://github.com/free-gen/EZ2Play/releases).

EZ2Play это минималистичный WPF лаунчер для запуска игр на вашем ТВ. В отличие от большинства игровых лаунчеров, где требуется добавлять игры и парсить обложки, здесь используются стандартные ярлыки Windows. Они уже содержат в себе все что нужно: путь к игре, при необходимости параметры запуска и собственную иконку (.ico), которая автоматически используется как изображение в интерфейсе.

От запуска лаунчера до старта игры всё взаимодействие происходит без клавиатуры и мыши. Переключение дисплея, например с монитора на телевизор, доступно по нажатию одной кнопки, поэтому не нужно заходить в параметры Windows или использовать сторонние утилиты. Такой подход упрощает процесс запуска и делает ПК более удобным для игры на телевизоре с геймпадом.

## Возможности

- **Управление с клавиатуры или геймпада**
- **Обработчик кнопки Home** - закрытие игры с геймпада и возврат в лаунчер
- **Быстрое переключение экрана** - опция доступна при наличии двух и более экранов
- **Мультиязычность** - поддержка русского и английского языков
- **Никаких настроек** - просто поместите ваши ярлыки в папку `shortcuts`
- **Современный дизайн** - WPF-UI с прозрачностью + анимированные частицы

## Использование

1. Скачайте и запустите `EZ2Play.exe`
2. Поместите ярлыки (.lnk и .url) игр и приложений в папку `shortcuts` (создаётся автоматически)
3. Лаунчер готов к работе

## Кастомизация

EZ2Play поддерживает небольшую кастомизацию внешнего вида и звуков.

### Логотип запуска

Чтобы заменить логотип на заставке:

1. Подготовьте изображение `logo.png`
2. Поместите файл рядом с `EZ2Play.exe`

При наличии файла лаунчер автоматически использует его вместо встроенного логотипа.

### Звуковые пакеты

Вы можете заменить системные звуки лаунчера с помощью пользовательского звукового пакета.

#### Как создать sound.pack:

1. Создайте обычный `.zip` архив
2. Поместите в него следующие файлы:
   - `select.mp3` - звук перемещения
   - `action.mp3` - звук запуска
   - `abort.mp3` - звук возврата
   - `ambient.mp3` - фоновая музыка
3. Переименуйте архив в `sound.pack`
4. Поместите его рядом с `EZ2Play.exe`

> Файлы должны находиться в корне архива (не внутри папки).

Если `sound.pack` отсутствует или в нём нет какого-либо файла, используется встроенный звук по умолчанию.

## Параметры командной строки

```bash
EZ2Play.exe [параметры]
```

| Параметр       | Описание |
|----------------|----------|
| `--nosplash`   | Запуск без экрана-заставки |
| `--hotswap`    | Автоматически переключает дисплей при запуске. При закрытии автоматически переключает дисплей на исходный |
| `--nomusic`    | Отключает фоновую музыку |

### Примеры

```bash
# Запуск без заставки
EZ2Play.exe --nosplash

# Запуск с автоматическим переключением дисплея
EZ2Play.exe --hotswap

# Запуск без фоновой музыки
EZ2Play.exe --nomusic

# Комбинация
EZ2Play.exe --hotswap --nosplash --nomusic
```
